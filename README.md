# 달서구 · 대구광역시

> 달서구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-daegu-dalseo.github.io/**

달서구는 대구광역시 8개 자치구 가운데 인구가 가장 많은 곳으로, 성서산업단지의 공장 지대와 두류·상인·월배의 대규모 아파트 주거지가 나란히 공존하는 산업·주거 복합 지역이다. 구 이름 '달서(達西)'는 1938년까지 존재했던 경상북도 달성군 달서면에서 따온 것으로, '달구벌의 서쪽'이라는 지역 정체성을 그대로 담고 있다. 진천동 입석과 월성동·상인동 일대에서 청동기시대 유물이 대거 쏟아져 나와 '선사시대 대구의 수도'로 불릴 만큼 오래된 삶의 터전이기도 하다. 오늘날에는 두류공원의 이월드와 83타워, 대구수목원, 대구문화예술회관 등이 모여 대구 시민의 대표적 문화·여가 공간으로 자리 잡았다. 첨단 제조업과 활발한 생활 인프라, 풍부한 녹지가 어우러져 '일하고 쉬고 즐기는' 자족형 생활권을 이루는 것이 달서구의 얼굴이다.
